# Evidence

Capturas y evidencia versionada de los experimentos. Vive dentro del repo, junto
a la bitácora, para que cada afirmación sea rastreable a su prueba.

## Convención de nombres

`exp-<NNN>-<descripcion-corta>.png`

Ejemplos:
- `exp-001-selfheal-pods.png`
- `exp-001-events-timestamps.png`

## Pendiente

Se copian aquí las capturas que ya tomamos de EXP-001 y las renombramos según la
convención. Luego `git add evidence/ && git commit && git push`.
